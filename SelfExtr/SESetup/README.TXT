WinZip Self-Extractor is a companion product for WinZip, an
award-winning utility that brings the convenience of Windows to
the use of ZIP files.  For program details please see the
online documentation or the WINZIPSE.TXT file.

*** Installation ***

    Installation from a floppy disk:  If you received WinZip
    Self-Extractor on a disk from Nico Mak Computing, Inc. please
    follow the installation instructions on the disk label.

    Installation the evaluation version:  The evaluation version
    of WinZip Self-Extractor is distributed as a self-extracting
    ZIP file.  To install it, run the self-extracting ZIP file.
    For example, if you received the program in the self-
    extracting ZIP file downloaded as C:\DNLOAD\WZIPSE21.EXE,
    activate the Program Manager, choose Run from the File menu,
    type C:\DNLOAD\WZIPSE21, and press the Enter key.  The SETUP
    program will automatically be run.

*** Requirements ***

    WinZip Self-Extractor requires Windows 95, Windows NT 4.0, or
    Windows 3.1 and a program like WinZip or PKZIP to create ZIP
    files.  See the documentation for details.

*** Documentation ***

    For full documentation press the Help button while running
    WinZip Self-Extractor.
